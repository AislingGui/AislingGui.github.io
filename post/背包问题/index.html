背包问题

2020-10-29
算法
times read

Contents

    • 0-1背包
    • 完全背包

背包问题是一种组合优化的问题，包括01 背包，完全背包，多重背包。

0-1背包

问题描述：有n种物体，物体j的体积为wj,价值为wi，有一个体积限制V。每一种物体只有1个，只能选或不选，使得背包物体的价值尽量大。状态设计:

1

dp[i][j] := 已经考虑了 [0..i-1]  前 i 个物品，占用了 j 空间，所能取得的最大价值。

转移方式：

1
2

dp[i][j] = dp[i - 1][j] 当前物品不选
           dp[i - 1][j - v[i]] + w[i] 当前物品选，j - v[i] 要大于等于0

初始化所有状态置为0即可。

0-1背包空间优化：

用滚动数组来保存状态，从大到小推

dp[j] = max(dp[j], dp[j - v[i]] + w[i]) // j 从大往小推

完全背包

有 n 种物品，物品 j 的体积为 vj，价值为 wi，有一个体积限制 V。每种物品有无限个，此问题称为完全背包问题。

由于体积有限制，每个物品其实最多只能取 V / v[i] 个。
